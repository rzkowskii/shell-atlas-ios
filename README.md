# Shell Atlas

Shell Atlas teaches Linux command-line basics through a star-map campaign, quick practice, and a safe simulated terminal.

[Visit the official site](https://shellatlas.app/)

## How It Works

Shell Atlas turns terminal basics into short iOS sessions. You follow a campaign map, unlock command stars, practice with lessons and puzzles, and review old commands so they stay familiar.

## What You See

- Campaign map: each command is a star you unlock along a learning path.
- Lessons: short explanations for beginner command concepts.
- Practice: command fragments you assemble in a safe puzzle screen.
- Review: old commands come back before they fade.
